# 🍃 EcoKafe - AI-Powered Sustainable Cafe Recommendations

EcoKafe is a modern SaaS platform designed to connect conscious consumers with sustainable cafes. It leverages Artificial Intelligence to recommend eco-friendly dining options and provides real-time surplus food alerts to drastically reduce food waste.

## 🚀 Key Features

* **🤖 AI-Powered Smart Search:** Integrated with **Google Gemini 1.0 Pro AI** to understand natural language queries and filter cafes based on real sustainability metrics, location, and user preferences.
* **🍱 Surplus Food Alerts:** A live broadcast system where cafe owners can list surplus food at discounted prices at the end of the day, helping users save money and the planet.
* **👑 Role-Based Dashboards:** * **User:** Track Eco-points, view saved cafes, and claim active surplus deals.
  * **Cafe Owner:** Publish live inventory alerts and track food waste recovered.
  * **Admin:** Verify new cafe registrations and monitor overall platform impact (CO2 offset, waste saved).
* **⚡ Modern Tech Stack:** Built on the latest **Next.js 14/15 App Router** for lightning-fast Server-Side Rendering (SSR) and seamless client-side navigation.

## 🛠️ Technology Stack

* **Frontend Framework:** Next.js (React.js)
* **Styling:** Custom CSS & Tailwind principles
* **AI Integration:** `@google/generative-ai` (Gemini API)
* **Routing:** Next.js App Router (`next/navigation`)
* **State Management:** React Context API & LocalStorage (Mock DB)
* **Data Visualization:** Chart.js & `react-chartjs-2`
